What is AI Prompting

AI prompting involves giving clear and purposeful input to an AI tool in order to guide its response. The quality of a prompt can shape how useful or accurate the AI's output is.

Effective prompting allows you to guide AI tools in ways that support the writing process without replacing critical thinking or original work.

How To Write A Good Prompt

  1. State Your Goal
    • Summarize in one sentence what you want (e.g., “Translate this text into layman’s terms”).
    • Mention why (e.g., “For middle-school readers”).
  2. Give Context
    • Include or describe the source material (text, data, code).
    • Specify audience and tone (e.g., “Explain as if to a non-technical manager”).
  3. Structure the Prompt
    • Instruction: One imperative verb + task (e.g., “List,” “Compare,” “Generate”).
    • Context/Reference: Paste or describe relevant passages or data.
    • Requirements: Format (e.g., bullet points, JSON), length limits, style guidelines.
  4. Be Specific
    • Avoid vague words like “Explain well.” Instead, say “Explain in ≤150 words, using three headings.”
    • Define any acronyms or specialized terms.
    • Clarify edge cases (“If no data is available, state ‘N/A’”).
  5. Specify Output Format
    • Outline formatting rules (“Use Markdown headings,” “Return a numbered list,” “Respond in JSON with keys ‘title’ and ‘summary’”).
    • Note language/style (“Formal tone,” “Active voice,” “American English”).
  6. Anticipate Pitfalls
    • Narrow the scope to prevent drift (“Focus only on economic factors, not social ones”).
    • Tie pronouns to nouns (“‘It’ refers to the software license”).
  7. Test & Refine
    • Run the prompt, compare output vs. expectations.
    • Tweak one element at a time (e.g., adjust word count if it’s too long).

Ethical Use

Ethical use of AI in writing involves using the technology to support, not replace, a student's original thinking and voice. Students are expected to follow UWB’s academic integrity guidelines when using AI tools.

This includes properly revising AI-generated content, ensuring all work reflects their own understanding, and consulting instructors when unsure about appropriate use.
